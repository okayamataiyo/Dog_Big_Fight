ゲーム説明
2匹の犬を操作してそれぞれのしたい目標の手助けをしよう！片方の犬は骨を集めるために走りまわる！もう片方の犬は構ってほしくて骨を集める犬の邪魔をする！片方がスコア150点になったらゲーム終了!
